
🍕 MauiApp7 — Приложение для заказа пиццы
Это учебный проект на .NET MAUI, реализующий систему выбора, кастомизации и заказа пиццы с сохранением данных в JSON.

📑 Описание страниц (Окон)
Название окна	Техническое имя	Что делает
Меню	MenuPage	Главный экран. Список доступных пицц. При нажатии перекидывает в конструктор.
Конструктор	ConstructorPage	Выбор размера (Маленькая/Средняя/Большая), добавление ингредиентов и выбор количества. Цена обновляется на лету.
Корзина	CartPage	Список выбранных товаров. Можно удалить позицию или перейти к оформлению.
Оформление	OrderPage	Ввод данных клиента (Имя, Телефон, Адрес). После подтверждения сохраняет всё в историю.
🚀 Как пользоваться (Гайд)
Выбор пиццы: На главном экране нажми на любую пиццу.

Настройка:

Выбери размер (цена изменится автоматически).

Поставь галочки на доп. ингредиенты (сыр, бекон и т.д.).

Нажми "В корзину".

Проверка: Перейди в корзину (через кнопку или вкладку). Если передумал — нажми "Удалить".

Заказ:

В корзине нажми "Оформить заказ".

Введи свои данные в окне оформления.

Нажми "ПОДТВЕРДИТЬ".

Появится уведомление, и приложение вернет тебя в меню, очистив корзину.

💾 Где искать заказы (JSON данные)
Все заказы сохраняются в файл orders_history.json.

Если ты менял путь на прямой: файл появится прямо в папке с проектом (там, где лежат твои .cs файлы).

Если путь стандартный:

Windows: %LocalAppData%\Packages\MauiApp7_...\LocalState\

Android: Внутренняя папка приложения (доступна через Device File Explorer в Visual Studio).

🛠 Технологии
MVVM Community Toolkit — для связи логики и интерфейса (атрибуты [ObservableProperty], [RelayCommand]).

Dependency Injection — все страницы и ViewModel регистрируются в MauiProgram.cs.

JSON Serialization — для сохранения истории заказов.
